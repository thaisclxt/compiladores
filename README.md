# Compiladores

Repositório para a disciplina de Compiladores

## Tarefa em lex

Comando para compilar lex -> `lex tarefa.l`

Comando para compilar o arquivo gerado em c -> `gcc lex.yy.c -o tarefa -ll`

Comando para executar o arquivo gerado -> `./tarefa`
